*A Web-Based Application for Finding Everyday Meals with Taste and Nutrition*

TasteTrove is a web-based recipe discovery platform designed to help users find **healthy, tasty, and nutritious meals** with ease.  
The application focuses on simplicity, accessibility, and informed eating by providing recipes along with **nutritional insights** and an **interactive cooking experience**.
